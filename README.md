# mini-meeting

# 文件介绍
文件夹"netapi"为网络封装包</br>
文件夹"服务端"为linux-qt的服务端包</br>
文件夹"demo"是windows-qt的客户端</br>


# 功能
1. 注册、登录（账号密码、微信、qq）
2. 大厅、房间系统
3. 采集视频、音频
4. 编码音频、解码音频、压缩视频
5. 发送音视频
6. 播放视频、播放声音
7. 优化---多线程...
